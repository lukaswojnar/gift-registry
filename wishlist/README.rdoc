== README

Wishlist is an application where users can manage gifts on specific event.
You just need to create your account. Then you can create an event and add gifts you want from another users.
An e-mail will be send to all invited user. They need to sign up if they want to work with your list.
Invited user can choose which gift he or she will buy for you. They can also let you know if they come to this event.
All users can set a notification. An e-mail will be send to their e-mail (with details about event and with list of all gifts they should buy).
